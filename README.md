# 🕵️ Phantom AI – Forensic Sketch Generator

## 🔍 Description
Phantom AI is a web-based forensic tool that generates realistic suspect images from structured descriptions using Generative AI.

---

## 🚨 Problem Statement
Manual suspect sketching is slow, subjective, and inconsistent. Investigators need a faster and more reliable way to visualize suspects based on descriptions.

---

## 💡 Solution
This system takes structured inputs such as age, gender, facial features, and distinguishing marks, and generates a realistic suspect image using AI.

---

## ✨ Features
- Structured suspect input
- AI-generated images
- Suspect profile card
- Case ID system
- Match verification
- Clean UI

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Generative AI (Pollinations API)

---

## ▶️ How to Run
1. Download or clone the repo
